# Um-Mapa-do-Tesouro

Sistema desenvolvido para gestão de produção agrícola de laranja.

Funcionalidades:

- cadastro de produção
- listagem de produções
- cálculo de perdas
- cálculo de produtividade

Tecnologias utilizadas:

Python
Estruturas de dados
Listas
Tuplas
Dicionários

Execução:

python src/main.py
