Sistema de gestão de produção de laranja.

Objetivo:
Auxiliar produtores rurais a registrar e analisar dados de produção da citricultura.

O sistema permite:

- cadastrar produções
- listar produções
- calcular perdas
- calcular produtividade

Tecnologias utilizadas:

Python
Listas
Tuplas
Dicionários
Manipulação de dados em memória
